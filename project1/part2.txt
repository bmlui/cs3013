1. What is the root directory? What is the command to change the current working directory to the root directory?
The root directory is /. The command to change the working directory to the root is cd /.

2. What is the command to create an empty file with the name foo.txt?
touch foo.txt

3. What is the command to rename the file foo.txt to bar.txt.
mv foo.txt bar.txt

4. What is the command to find location of the binary executable file for the ls utility? What is the output of running that command?
pwd ; the output of running that command prints the current working directory. For my case, this resulted in /home/blui/Documents/Git/cs3013.

5. What is the command to print the username of the current user? What is the output of running that command?
whoami ; the output prints the username of the current user. My output was blui.

6. What are shell environment variables? What command will print out the current environment variables (note, you should not provide the output of this command)?
Shell enviroment variables are dynamic values stored by the system that affect how different applications run. Use the printenv command to print out he corrent enviroment variables. 

7. What is the command to print the name of the current shell? What is the output of running that command?
The command is echo $SHELL or printenv SHELL ;  The output is /bin/bash . 

8. What is the command to print out the kernel version of your Ubuntu OS? What is the output of running that command?
The command is uname -r ; The output is 5.15.0-58-generic .
